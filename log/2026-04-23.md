# v1.0.0 - 初始版本

> 日期: 2026-04-23

## 项目概述

全栈视频流媒体服务应用，支持视频上传、HLS转码流媒体播放、用户认证与二维码分享。

## 技术栈

- **后端**: Node.js + Express
- **前端**: Vue 3 + Vite + Element Plus
- **数据库**: LowDB (JSON 文件存储)
- **认证**: JWT + bcryptjs
- **视频处理**: fluent-ffmpeg (HLS 转码)
- **文件上传**: Multer
- **二维码**: QRCode

## 核心功能

1. **用户系统**: 注册、登录、JWT Token 鉴权
2. **视频上传**: 支持 mp4 等格式上传，Multer 处理文件接收
3. **HLS 转码**: 自动将上传视频转码为 HLS (m3u8 + ts) 格式，支持流媒体播放
4. **视频管理**: 视频列表查看、删除
5. **二维码分享**: 生成视频链接二维码，支持扫码观看
6. **路由守卫**: 前端路由鉴权，未登录自动跳转登录页
